# Hebrew–English Nutrition Glossary

Load only when a term is unclear. Israeli colloquial food terms included.

## Macros & metrics
- חלבון = protein · פחמימה = carbohydrate · שומן = fat · קלוריות = calories
- מאקרו = macros · אחוז שומן = body-fat % · מסת שריר = muscle mass · משקל = weight
- היקפים = measurements/circumferences · BMR / חילוף חומרים = metabolic rate

## Meals
- ארוחת בוקר = breakfast · ארוחת צהריים = lunch · ארוחת ערב = dinner
- ארוחת ביניים = snack/intermediate · ארוחת לילה = night meal · נשנוש = snacking

## Foods
- חזה עוף = chicken breast · הודו = turkey · קציצות = meatballs · טופו = tofu
- ביצה (קשה) = (hard-boiled) egg · שקשוקה = shakshuka · קוטג' = cottage cheese
- גבינה צהובה = yellow cheese · גבינת שמנת = cream cheese · יוגורט = yogurt
- אבוקדו = avocado · טחינה = tahini · שקדים = almonds · אגוזי מלך = walnuts
- אורז מלא = brown rice · פיתה קלה = light pita · לחם מלא = whole bread
- שיבולת שועל = oats · עדשים = lentils · אפונה = peas · בטטה = sweet potato
- מלפפון = cucumber · עגבנייה = tomato · גמבה = bell pepper · צנונית = radish
- תרד = spinach · ברוקולי = broccoli · כרובית = cauliflower · שעועית ירוקה = green beans
- חציל = eggplant (**never suggest**) · דג / סלמון / טונה = fish/salmon/tuna (**never suggest**)

## Training
- אימון = workout · אימון כוח = strength training · אירובי = cardio
- לפני אימון = pre-workout · אחרי אימון = post-workout · יום מנוחה = rest day

## Goals
- ירידה במשקל / חיטוב = fat loss/cut · שמירה = maintain · עליה במסת שריר = build muscle
- מטרה = goal · יעד = target
